# Programação I — ECO 2026 · Entregas

Repositório oficial de entregas de exercícios e do projeto semestral da disciplina **Programação I** — IFC São Bento do Sul — ECO 2026.

---

## Estrutura do repositório

```
prog1-eco-2026-entregas/
├── exercicios/
│   ├── aula-01/
│   ├── aula-02/
│   ├── aula-03/
│   └── ... (uma pasta por aula)
└── projetos/
    ├── equipes/
    │   ├── equipe-01/
    │   ├── equipe-02/
    │   ├── equipe-03/
    │   └── equipe-04/
    └── produto-final/
```

---

## Como entregar exercícios

1. **Clone** este repositório (ou faça fork, conforme instrução da professora).
2. Dentro da pasta da aula correspondente, **crie uma pasta com o seu nome** (ex.: `exercicios/aula-01/joao-silva/`).
3. Coloque seus arquivos de código dentro da sua pasta.
4. Faça `git add`, `git commit` e `git push`.

> **Padrão da mensagem de commit:**  
> `Entrega exercicios aula-NN - Nome Completo`

---

## Como entregar o projeto semestral

- Cada equipe tem uma pasta em `projetos/equipes/equipe-XX/`.
- Trabalhe **apenas na pasta da sua equipe**.
- O código final integrado será reunido em `projetos/produto-final/` ao final do semestre.

> **Padrão da mensagem de commit:**  
> `[Equipe-XX] descricao do que foi feito`

---

## Comandos rápidos

```bash
# Clonar este repositório
git clone https://github.com/wilcilene/prog1-eco-2026-entregas.git
cd prog1-eco-2026-entregas

# Atualizar antes de começar a trabalhar
git pull

# Fluxo de entrega
git add .
git commit -m "Entrega exercicios aula-01 - Seu Nome"
git push
```
