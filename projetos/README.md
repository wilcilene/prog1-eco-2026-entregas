# Projetos — Programação I ECO 2026

Esta pasta contém o projeto semestral desenvolvido em equipes.

## Organização

```
projetos/
├── equipes/
│   ├── equipe-01/   ← cada equipe trabalha aqui
│   ├── equipe-02/
│   ├── equipe-03/
│   └── equipe-04/
└── produto-final/   ← integração final ao final do semestre
```

## Regras do projeto

- Cada equipe deve trabalhar **apenas na sua pasta** (`equipes/equipe-XX/`).
- Cada pasta de equipe deve ter um `README.md` com: integrantes, descrição da parte do sistema, instruções de uso e plano de integração.
- A integração final será feita junto com as equipes ao final do semestre em `produto-final/`.

## Padrão de commit para o projeto

```
[Equipe-XX] descricao do que foi feito
```

Exemplo: `[Equipe-01] implementa tela de login`

## Ideia do projeto semestral

Cada equipe será responsável por uma parte de um sistema real e útil. As partes serão conectadas ao final do semestre, formando um único produto entregavel que poderá ser utilizado na vida real.

> Detalhes do sistema, divisão de equipes e requisitos de cada parte serão publicados pela professora no início do semestre.
